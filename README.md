📖 Course Overview
This repository contains my solutions for the EPI module. The course introduces fundamental programming concepts, algorithmic thinking, and data manipulation using Python.

📬 Pro Tip: I highly recommend reading the Assignment PDF for each week before looking at the code. The PDFs explain the underlying logic and specific constraints for each task.

🛠 Core Topics & Skills
The assignments in this repository cover the following progression:

1. Basic Syntax & Control Flow
Variables & Data Types: Working with Strings, Integers, Floats, and Booleans.

Control Structures: Mastering if/else logic and loop variants (for, while).

Functions: Defining reusable code blocks and understanding scope.

2. Data Structure Manipulation
Lists: Slicing, indexing, list comprehensions, and nested lists.

Dictionaries: Key-value pair manipulation, frequency maps, and building structured datasets.

Tuples & Sets: Understanding immutability and uniqueness.

3. Graphics & Visualization
Turtle Graphics: Learning procedural drawing and geometric patterns through the turtle library.

Matplotlib: (Often used alongside NumPy) for plotting mathematical functions and data.

4. Scientific Computing & Libraries
NumPy: Array manipulation, linear algebra, and efficient numerical calculations.

File I/O: Reading from and writing to .txt and .csv files to build simple databases.

Exception Handling: Using try/except to build robust programs.
